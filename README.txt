Сглаживание по среднему производить в окне M. M читается из входных аргументов.
Сглаживание с окном M означает что в ячейку n выходного набора должно записаться среднеарифметическое значение входных данных от n-M до n+M
Считать что n>>М.
Пример запуска вашей программы: ./smooth Test.txt 10. 
Выполнять следует на языке C++. Лучше использовать библиотеку stl. 
Программа должна без крашей работать с тестовым набором, а так же с бОльшим и меньшим количеством входных данных. 
Выводить на экран информацию об ошибке в случае не корректных аргументов. 
Сглаженый массив вывести в выходной файл.
Текст программы должен быть оптимально написан с точки зрения производительности, быть аккуратным, легко читаться. 
Важный пункт: выбирая между понятностью и производительностью выбирать первое.
Основная задача сглаживания имеет несколько решений, с разной временной сложностью. Следует указать временную сложность представленного вами алгоритма.
